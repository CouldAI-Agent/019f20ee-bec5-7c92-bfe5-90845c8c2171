# Zepto Clone (ज़ेप्टो क्लोन)

यह एक 10-मिनट ग्रॉसरी डिलीवरी ऐप (Zepto) का क्लोन है। इसे फ्लटर (Flutter) का उपयोग करके बनाया गया है। इस ऐप में होम स्क्रीन, कैटेगरीज, कार्ट और प्रोफाइल जैसी प्रमुख स्क्रीन शामिल हैं।

## मुख्य विशेषताएँ (Features)
- **होम स्क्रीन (Home Screen):** आकर्षक बैनर, कैटेगरी ग्रिड और डेली एसेंशियल प्रोडक्ट्स।
- **कैटेगरीज (Categories):** सभी उपलब्ध सामानों की सूची।
- **कार्ट (Cart):** खरीदे जाने वाले उत्पादों का प्रबंध।
- **प्रोफाइल (Profile):** उपयोगकर्ता की जानकारी, ऑर्डर्स, पते और सपोर्ट।

## तकनीक (Tech Stack)
- **फ़्रेमवर्क:** Flutter (Dart)
- **डिज़ाइन:** Material 3, Responsive UI

## कैसे चलाएं (How to Run)
1. रिपॉजिटरी क्लोन करें।
2. टर्मिनल में `flutter pub get` रन करें।
3. ऐप चलाने के लिए `flutter run` का उपयोग करें।
